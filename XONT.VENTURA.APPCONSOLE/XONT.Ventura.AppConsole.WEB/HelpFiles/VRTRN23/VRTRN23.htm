CONSOLIDATE PURCHASE ORDER

 

This component is used to consolidate the suggested orders which have been already created in the system. In general this component is used by distributors. The initial load screen will display the ordered products and quantities by each executive.

1st Screen

  Initial screen will display the suggested products and quantities followed by the total value at the bottom. If there are no suggested order found, message will be displayed as following that there are no suggested orders found.

 

v  There are different Options associated with the 2nd screen.

Add New suggested order

  Click New Button

Save an order

  Click Save Button

Consolidate an order

  Click Consolidate Button

Exit from Consolidate purchase order

  Click Exit Button

Adding New ENTRY TO AN ORDER

  This option will allow adding new order/ new entries to an existing order.

  When clicking on the New button, following screen will be displayed where the user can enter quantities against relevant products for each executive.

Please note that the user can add quantities, only for the products which are attached to the executive.

  User can add quantities and click on the Add & Hide button in order to add the products to the existing suggested order.

  Save button enable the user to save the newly added records to the database.

 

Consolidating the suggested orders

  User can consolidate the suggested orders by clicking on the Consolidate button. When clicking on Consolidate button, following interface appears, where the user will be able to select the consolidation method, whether, Distributer/ Territory/ Rep.

  After selecting the consolidation type, user will be prompt to enter the remarks through the following window.

  After entering the remarks and tapping on OK button, suggested orders will be consolidated and create purchase orders.
