product Maintenance

 

This Component is used to maintain details related to product and product attributes. At the initial load screen all available locations will be listed in the grid.

1st Screen

v  By expanding the Selection Criteria panel, User will be able to filter the available products accordingly.

 

Selection could be based on product classifications or other product attributes. By unticking the active checkbox user can view the active and inactive records as well.

 

v  There are different Options associated with the Main screen.

 

Add New Product to the System

  Click New Button

 

Add New Product Based on an existing Product

  Select specific Product New Based On Button

 

Edit specific Product in the System

  Select Product Edit Button

 

Exit from Product Maintenance

  Click Exit Button

 

v  In the Product Creation and Editing Process following fields are mandatory.

  Product Code

  Product Description

  Classification Values of the Classification Codes

  Default Unit

  Preferred Supplier Code

 

 

 

 

 

Creating a new Product

A new product can be a completely new product or based on an existing product.

 

Creating A completely New product

 

 

 

  User can create a completely new location by clicking the New button.

 

  When clicking the new button, following interface appears and the user will be able to enter the product features.

 

 

 

 

 

 

2nd Screen

 

  Product Code: A unique code to identify the product and differentiate from the other products.

 

v  Product Details tab

  General Description:

  Description: Product description will be entered in this field.

  Alt Description: This will allow maintaining an alternate description for the product

 

  Price:

User can add the price and cost details of the product. Details such as Standard cost and Buying Price can be entered here.

 

  Classification:

  These fields will allow maintaining the classifications of the product.

  Each codes are selected from the predefined classification codes and values maintained in the classification code (Refer: Classification Code Maintenance).

  Unit of measure:

This will have two fields Def Unit and Alt Unit which can be used to store the Unit of Measures of the product.

 

  Conversion Factor:

Conversion factor will be used to maintain an alliance between both units of measures.

 

  Tax Details:

Product tax details will be maintained here. Drop down menu will allow selecting the Tax types which will be maintained in Tax maintenance (Refer: Tax Maintenance).

Status flag will allow enabling or disabling the Tax for the product.

  All taxes will be applied only if VAT Apply flag is enabled in Territory Control Maintenance. (Refer: Territory Control Maintenance.)

 

v  Stock Tab:

 

This tab will allow maintaining details regarding the product stocks. Details such as Stock details, Volume and Part attribute details will be maintained here.

 

v  Other Details Tab

 

This tab will allow to maintain additional details of the product. Detailes such as Product Type, Product Settings, Bar Code and Lot Number Details can be maitained here.

 

 

 

  Advance Options

 

By tapping on the Advance Options button in the Other details tab, user can navigate to the Advance options window. User can Enter the purchasing details, Sales and export details, Report criterias and other details such as raw meterial flag and expired product handling in the Advanced options.

After entering relevant details, user can click OK to save the product or Cancel to discard the specifications and return to the 1st Screen.

Creating A NEW PRODUCT BASED ON AN EXISTING product

 

To create a new Product, based on an already available Product, Select corresponding Product and click New Based On button.

 

The following interface is displayed where all details are copied from the Based on code except the Product Code which is a Unique Code for each product. User can edit necessary fields and save or discard his specifications.

 

 

 

EDITING a product

 

  User can edit a product attributes by selecting a product from the grid and tapping on the Edit button.

  Following interface is displayed after selecting the edit option.

  The user can change all attributes except the Buying Price and the Product Code, which is a unique code for each product. User can edit the relevant fields and save or discard his specifications.
