WEEKLY AND MONTHLY TARGET MAINTENANCE

 

This Component is used to maintain value and volume target for sales rep weekly. By adding up all weekly value targets calculate as monthly target.

At the initial load 1st screen user will be required to select a territory, Executive Type, Executive and Target type.

1st Screen

 

  After selecting the Territory and Executive Type load Executives which is relevant to the selected territory and Executive Type.

  After selecting the target type (Volume or Value) and select the Monthly option display the Year and Month selections as following Screen (2nd Screen)

2nd Screen

  If user select Weekly option then user should have to select the week also. Week should be able to capture form calendar.

 

    After click on list button display the executive which is relevant to territory and executive type.

  User can allow entering target Value or Volume in grid text box by using Enter key.

  Clicking on OK button after adding target all target will save to relevant tables and display the message which is saying Data Upload Successfully.

 

 

  If user clicking on Cancel button discard all his specification.

 

  If user Clicking on Exit button display confirmation message which is saying Are you want to Close this window and if user select Yes then close component if select No keep that screen as it is.

 

 

 

 

  Enter the Targets correctly since that it will be used in many reports.

  First user should have to maintain correct week according to calendar

 

 

  Do not enter any unwanted Targets.
