Vehicle Maintenance

 

This Component is used to maintain details related to Vehicles in the system.

At the initial load screen all available Vehicles will be listed in the grid.

1st Screen

 

v  By expanding the Selection Criteria, User will be able to filter Vehicles accordingly.

 

v  There are different Options associated with the Main screen.

 

Add New Vehicle to the System

  Click New Button

 

Add New Vehicle Based on an existing Vehicle

  Select specificVehicle New Based On Button

 

Edit specific Vehicle in the System

  Select Vehicle Edit Button

Exit from VehicleMaintenance

  Click Exit Button

 

 

v  In the Vehicle Creation and Editing Process following fields are mandatory.

  VehicleNumber

  Description

 

Creating a new Vehicle

Creating A completely New Vehicle

  User can create a completely new record of aVehicleby clicking the New button.

 

  When clicking the new button, following interface appears and the user will be able to enter the Vehicledetails.

 

2nd Screen

  VehicleNumber:A unique code which allows the user to identify the Vehicle and differentiate from other Vehicles.

 

  Registration Number: Allows the user to enter the Vehicleregistration number.

 

  User can also enter details such as Make, Model, Engine Number, Vehicle Chessie Number, Loading Capacity, Wight limit, Engine Capacity and Description.

 

  User also can select the Territory, warehouse and location if exists.

 

  User also can enter the analysis category and values respectively, if exists.

 

  Status of the vehicle determines whether the vehicle is active or inactive.

 

After entering relevant details, user can click OK to save the Vehicle or Cancel to discard the specifications and return to the 1st Screen.

Creating A NEW Vehicle BASED ON AN EXISTING Vehicle

 

To create a new Vehicle, based on an existing Vehicle, Select corresponding Vehicleand click New Based On button.

 

The following interface is displayed where all details are copied from the Based on code except the VehicleNumberand Registration Number. User is allowed free to edit necessary fields and save or discard his specifications.
