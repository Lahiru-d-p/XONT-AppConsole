EXECUTIVE TARGET UPLOAD

 

This Component is used to maintain Quantity, value and volume target for sales rep monthly.

At the initial load 1st screen and user will be required to select a Territory, Period, and Upload Type.

User should have to browse the relevant Excel File also.

1st Screen

User should have to maintain relevant target in Excel Sheet.

 

Following is the sample target Excel sheet format. (Columns Executives , Rows Products)

 

 

In Excel book name must be selected Territory code . And Excel sheet name must be a relevant target upload Year and the Period.

eg:- 2012-12

If user wants to upload target week wise then week number should have to include as sheet name.

eg :- 2012-12-1 (week 1)

 

  After browse the excel sheet user must have to validate that excel sheet by clicking on Validate button.

  If there is any errors, conflicts in excel sheet display following message and User can view error sheet by clicking on Log button.

 

 

  Display screen with error like as following screen

 

 

If there are no errors in Excel file OK button will activate.

All targets will save in to relevant table by clicking on OK button and display the Successfully uploaded message.

 

If user trying to upload the target for already entered targets then system give confirmation message saying Targets are already exists, Do you want to overwrite . If user click on yes button then targets are overwrite.

 

  If user Clicking on Exit button display confirmation message which is saying Are you want to Close this window and if user select Yes then close component if select No keep that screen as it is.
