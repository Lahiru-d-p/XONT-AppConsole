WEEK DEFINITION

 

This Component is used to maintain week definitions in the system. The weeks which are maintained here will de used in many other transactions.

Ex: Commission process

Initial screen will allow the user to enter week definition details.

 

1st Screen

 

  Module: Requires the user to select the module code through the prompt.

 

  When User selects the Year and Period from the dropdown, available week definitions for the selected period will be loaded as shown below (If exist).

 

 

 

 

 

 

 

 

  If user wants to enter new week definition, user can select the corresponding year and period from the dropdown and enter below details.

 

  Week Number: Allows the user to select the Number of the week from the drop down.

 

  Description: Allows the user to enter the description if exists.

 

  Start Date: Requires the user to select the start date of the selected week from the calendar control.

 

  End Date: Requires the user to enter the End date of the selected week from the calendar control.

 

  After entering above details, user can click on the Add button to add the defined week.

 

  User should tap on the OK button to save his specifications or Exit to exit the component.

 

  User can also remove a week definition by selecting the record and clicking on the Remove button.
